Login Module

Registered users are able to login in the system by correctly entering there User ID and Password. If you are not registered in our system, you may click on the "register" bottom.

Register

User ID - Please enter your person id.
FirstName - Please enter your first name.
LastName - Please enter your last name.
Password - Create a password of your own.
Re-Password - Has to be the same as above.
Class - Please choose your class.
Administer - Able to access and edit all users' information and records.
Doctor - Able to search records about his/her patients.
Patient - Able to search records about themselves.
Radiologist - Able to search records about his/her patients and upload radiology images.
Address - Please enter your address.
Email - Please enter your email.
Phone - Please enter you phone number.

Welcome Module

After login, user are able to see this page. In this page, there is an account information displayed on the top of the page. Also, you are albe to edit your own account information. You may just simply enter the blanks that you wish to modify to upload your information.

There is also a search option for each user. However, for the security issue, you are only able to search records about you.

Also there are different function options according to your class.

Administer - Able to use "Search" (Search all records), "Management" (edit and manage all acounts), "Data Analysis" (Generate a data anaysis)
Doctor - Able to use "Search" (Search all his/her patients' records)
Patient - Able to use "Search" (Search all his/her records)
Radiologist - Able to use "Search" (Search all his/her patients' records), "UploadPic" (Upload radiology images to records), "UploadInfo" (Upload radiology records)

UploadPic Module

This module is used by radiologis to upload radiology images.

The user can simply click on "Browse..." to select one or more images, and click "Upload".

UploadInfo Module

This module is used by radiologis to upload radiology records.

The user are able to fill in all blanks provided about records and click on "Process" to upload records.

Manaegement Module

This module is used by administer to edit or manage all accounts.

There is a table of all users and their information. The administer can simply enter the person_id which he/she wants to edit in the box below. After click on "Managnment", the administer is able to edit and manage all acount by fill in boxes which he/she wants to edit.

Report Module

This module is used by administer to generate a report about specified diagnosis.

After the administer specified a diagnosis and time period, the administer may click on "Report". After that it will generate a table of all information of patient about that diagnosis.

Search Module

This module is able to be used by users in all class. It is able to search records of some specified search conditions.

In this page, you can choose from several search conditions.

First & Last Names - You may enter some names according to your search.
Diagnosis - You may enter key words seperated by "," about diagnosis.
Test Type - You may enter key words seperated by "," about test type.
Description - You may enter key words seperated by "," about description.
Time Period - You may enter time period about your search in "DD-MM-YY" format.

Data Analysis Module

This module is used by administer to generate a data analysis report.

The administer may select some specified conditions such as patient id, test type and time period and the administer may also choose to let the report to group by yearly, monthly or weekly.
